# 📦 Delivery Rider Mobile Application

A clean, modern, and professional mobile application built with **React Native (Expo)** designed for delivery riders to manage and track their pending orders efficiently.

## ✨ Features

* **Real-time Order Tracking:** View all pending orders with detailed customer information fetched dynamically from the Laravel backend.
* **Professional UI:** Clean, responsive, and pixel-perfect card design using **Ionicons**.
* **Status Management:** Quick "Mark as Delivered" action which updates the MySQL database instantly.
* **Optimized Layouts:** Properly aligned multi-line address and customer notes fields for easy reading on the go.

---

## 🛠️ Tech Stack

* **Frontend:** React Native, Expo
* **Icons:** @expo/vector-icons (Ionicons)
* **Backend:** Laravel (E-commerce Web System API)
* **Database:** MySQL
